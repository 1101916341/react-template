## 特点

- 多 tab 页签 ➕ 右键菜单，提升效率
- Redux 状态管理 ➕ 持久化
- 封装实用 axios 请求
- 菜单页面路由权限控制

### 使用命令行

```bash
$ npm install -g typescript
$ git clone https://github.com/1101916341/react-template.git
$ yarn install
$ yarn start
```

### 提交命令

```bash
$ git add .
$ git commit -m '(fix|feat): 提交信息' 
$ git push

具体的commit信息，需要参考.cz-config.js文件
```
